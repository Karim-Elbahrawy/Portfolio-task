# AI Agent Project Brief: Create the LSS 1003 E-Portfolio Website

## Objective
Create a complete **7-page e-portfolio website** for the **LSS 1003 – Life and Future Skills** course, based strictly on the assignment instructions, rubric, checklist, and submission requirements.

This portfolio must be:
- personalized to the student
- visually professional
- fully aligned with the rubric
- complete with all required sections
- structured so the student can easily replace placeholders with their own screenshots, certificates, text, and video link

---

## Critical Goal
The website should be built to maximize marks by satisfying all major rubric categories:

1. **Explanation & Description / Depth of Analysis (30%)**
2. **Data Collection / Evidence (30%)**
3. **Productivity & Initiative (5%)**
4. **Clarity, Flow, and Mechanics (5%)**
5. **Presentation & Visual Impact (25%)**

The final result must feel complete, organized, polished, and easy to navigate.

---

## Core Assignment Facts
- **Course Code:** LSS 1003
- **Course Title:** Life and Future Skills
- **Assessment:** Assessment 2 – Portfolio / E-Portfolio
- **Weight:** 30%
- **CLOs:** 2, 3, 4
- **Submission deadline:** **17 April 2026, 11:59 PM**
- **Submission method:** submit the **e-portfolio link** through Blackboard using the official submission form

---

## Output Required from the AI Agent
The AI agent should create:

1. A complete **portfolio website structure**
2. All **7 required pages**
3. Strong **section-by-section written content drafts**
4. Clear **placeholders for personal evidence**
5. Clear **placeholders for screenshots, certificates, posters, and video**
6. A clean, modern, academic visual style
7. A version that is easy to publish using a website builder or code

If the agent is coding the website, it should generate:
- semantic HTML structure
- clean CSS styling
- responsive layout
- reusable page sections/components where appropriate

If the agent is generating the content only, it should still organize everything page by page in final-ready format.

---

## Non-Negotiable Rules
The AI agent **must not miss any of the following**:

- The website must contain **exactly 7 pages**
- The navigation must be simple and clear
- The **Home page must include an AI-generated future-career image**
- The **Effective Learning Techniques page must NOT use AI-generated images**
- The **Conclusion page must include a 60–90 second reflection video**
- The video must show the **student’s face**
- The **Achievements & Certificates page must include 2 LinkedIn certificates**
- The **Weekly Tasks page must include all required weekly tasks/evidence**
- The website must be personalized to the student
- The structure must support high marks on the rubric
- The final design should look creative, polished, and professional

# Rubric-Driven Build Strategy

## To score highly in Explanation & Description / Depth of Analysis
The AI agent should:
- write detailed section intros
- include reflective language
- connect activities to learning and future goals
- avoid shallow one-line descriptions where deeper reflection is possible

## To score highly in Data Collection / Evidence
The AI agent should:
- include every required weekly task
- clearly mark where evidence goes
- organize artifacts neatly
- show that all evidence connects to the learning journey

## To score highly in Productivity & Initiative
The AI agent should:
- make the portfolio look thoughtful and creative
- show effort in layout and presentation
- add reflective captions where suitable

## To score highly in Clarity, Flow, and Mechanics
The AI agent should:
- maintain logical page order
- use clean headings and subheadings
- ensure grammar is polished
- make the site easy to scan and understand

## To score highly in Presentation & Visual Impact
The AI agent should:
- create a professional homepage
- include the required AI career image
- include the required video section
- use strong formatting and balanced visuals throughout
